The code used in the work presented in my individual report.

To get the results:
- Run GBRTBidding.py, this will get the predictions and evaluate the Linear bidding strategy. The predictions will saved in a file.
- Run LogisticRegressionBidding.py this will get the predictions and evaluate the Linear bidding strategy. The predictions will saved in a file.
- Run ORBTbidding.py, this will get the results from both files and applies the two ORTB formulas while tuning c.

To view the plots described in data exploration part:
- Run DataExploration.py
